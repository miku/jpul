Jobportal Widgets und API

Sie können aktuelle Angebote des Career Center Jobportals auch in Ihre Seite einbetten. Nutzen Sie dazu unser experimentelles Widget.

Das Widget finden Sie unter http://localhost:10000/job/index?v=embed

Als Beispiel für die Einbettung schauen Sie in den Quellcode dieser Seite.

Sie können das Widget anpassen, indem Sie in der URL Suchbegriffe übergeben, die die Resultate entsprechend einschränken. Folgende URL liefert das Widget mit den aktuellen Angeboten, in denen der Begriff "Sport" vorkommt.

http://localhost:10000/job/index?v=embed&q=sport

Eine weitere Möglichkeit aktuelle Jobangebote vom Jobportal zu exportieren ist über eine JSON-Schnittstelle. Probieren Sie sie aus: 

$ curl "http://localhost:10000/job/index?v=json&q=sport"
